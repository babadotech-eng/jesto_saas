3. Calculations
Auto-calculate live:
- encargos_trabalhistas_pct =
  provisao_ferias_pct +
  provisao_terco_ferias_pct +
  provisao_decimo_terceiro_pct +
  provisao_decimo_terceiro_ferias_pct

- encargos_sociais_pct =
  inss_patronal_pct +
  sat_rat_pct +
  salario_educacao_pct +
  sistema_s_pct +
  fgts_pct +
  fgts_rescisao_pct

- encargos_totais_pct =
  encargos_trabalhistas_pct + encargos_sociais_pct

- valor_encargos =
  salario * encargos_totais_pct / 100

- total_mensal =
  salario + valor_encargos + vale_transporte + vale_refeicao + convenio_medico

- valor_hora =
  total_mensal / carga_horaria_mensal

If carga_horaria_mensal is empty or zero, use 220.
